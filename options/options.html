Basics

How it works

  1. You can generata a preview by either pressing keyboard shortcuts (alt + A or alt + shift + A), or clicking extension button ().
  2. If preview exists, clicking extension button () just toggle it's visibility.

  • It's convenient if you need to read lots of articles.
  • Clicking extension () can disable the automation (it will work as manual mode).
